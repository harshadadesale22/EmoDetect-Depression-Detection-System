# EmoDetect: Depression Detection System

## Overview
EmoDetect is an NLP-based web application that detects depression from user text using a BERT model.

## Features
- Depression detection using BERT
- SHAP explainability
- React frontend
- Flask backend
- REST API integration

## Technologies Used
- React.js
- Flask
- Python
- BERT (Transformers)
- SHAP
- HTML
- CSS
- JavaScript

## Installation

### Backend

```bash
pip install -r requirements.txt
python app.py
```

### Frontend

```bash
npm install
npm start
```
